# jd2-selkies

**⚠️ Hacky JDownloader 2 on Selkies – NOT for production use! ⚠️**

Hacky attempt to run JDownloader 2 on the LinuxServer Selkies base image. It does work with some massaging but I have no idea how to build containers correctly. Might be an OK starting point for someone that does.

- [Base image](https://ghcr.io/linuxserver/baseimage-selkies:alpine322)
- [LinuxServer Baseimage Selkies docs](https://docs.linuxserver.io/images/docker-baseimage-selkies)
- [Selkies docs](https://selkies.io/docs/)
- [JDownloader](https://jdownloader.org/)
